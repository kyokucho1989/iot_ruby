# Rubyで作りたいWEBサービスとRubyの学習記録
## 作りたいWEBサービス「植物観察日記」
### 概要
　朝顔の観察日記のようなものをWEB上でできるようにする。
 はじめはWEBのみ。できたらIoTシステムを構築して温湿度センサーから自動的にWEBへあげるようにする。
### 要件
* ログイン機能

* 記録　（日付・写真・温度・湿度・コメント）

* +アルファ　（ツイッター連携・他人でも見れる・コメントのやりとり...）
* +IoTシステムによる自動投稿 （温湿度センサー・カメラから自動記録・投稿）
